# Repositório do GitHub

Um *repositório do GitHub* é um espaço de armazenamento onde você guarda os arquivos de um projeto código-fonte, documentação, imagens e qualquer outro recurso necessário.

## Para que serve?

- *Controle de versão:* registra todo o histórico de alterações do projeto, permitindo voltar a versões anteriores a qualquer momento.
- 
- *Colaboração:* várias pessoas podem trabalhar no mesmo projeto simultaneamente, propondo mudanças por meio de pull requests.
- 
- *Backup na nuvem:* seu código fica salvo remotamente, protegido contra perdas locais.
- 
- *Organização:* issues, projetos e wikis ajudam a gerenciar tarefas e documentar o trabalho.
- 
- *Compartilhamento:* repositórios públicos permitem que qualquer pessoa veja, use ou contribua com seu projeto a base do open source.

## Em resumo um repositório é o coração de qualquer projeto no GitHub: é onde o código vive, evolui e é compartilhado com o mundo.
